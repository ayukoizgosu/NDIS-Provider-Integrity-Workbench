# Methodology

This tool combines public enforcement records, public ABN register results, and the public ASIC company dataset to help an analyst review provider records in one place.

## What the workflow does

1. Collects published NDIS enforcement records.
2. Groups repeated actions into entity-level profiles.
3. Links records to public ABN and ASIC references where available.
4. Flags related later businesses with similar names for analyst review.
5. Lets an analyst create a case, record notes, and export a brief.

## What the workflow does not do

- It does not use internal NDIA claims or payment data.
- It does not make a finding of fraud.
- It does not prove phoenix behaviour.
- It does not replace investigative or legal review.

## Analyst use

This is an analyst support workflow. A user should always read the linked source material and apply judgement before escalating a case.
