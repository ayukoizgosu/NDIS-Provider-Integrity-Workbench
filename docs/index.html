Workflow

How Work Moves

  1. Search a person or provider from the record view.
  2. Create a case and pull the linked business context into the desk.
  3. Run a live register refresh to capture the latest public business status.
  4. Review the agent-prepared summary, evidence, and judgment steps.
  5. Export a case brief with source links and review notes.

Boundaries

Current Scope

  • Uses public records only.
  • Supports review and documentation, not automatic decisions.
  • Uses ABR data and ASIC public datasets, not paid director extracts.
  • Ships with static sample data for demonstration and hosted review flows.
