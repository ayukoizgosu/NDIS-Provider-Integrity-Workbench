What It Does

Pilot-Safe Analyst Workflow

Current capability

  • NDIS enforcement record ingestion and normalization
  • ABN and ASIC public-register enrichment
  • Conservative related-business lead generation
  • Case desk with notes, attachments, and activity history
  • Live ABR refresh and ASIC snapshot checks
  • HTML brief export for supervisor review

Current boundary

  • No claims, payment, or bank data
  • No automated enforcement decisions
  • No paid director extracts
  • No internal NDIA or Commission integrations
  • No production identity, hosting, or accreditation controls in this repo

Review Flow

How A Case Moves

  1. Search a provider or person from the public-record view.
  2. Create a case and pull the linked business evidence into the desk.
  3. Run a live register refresh to capture ABR status and latest ASIC dataset context.
  4. Review the agent draft, completed checks, evidence used, and analyst-judgment steps.
  5. Export a brief for supervisor review or escalation.

Positioning

What To Show Government Buyers

This repo is positioned as a provider-integrity workbench and public-record analyst support tool. It is suitable for pilot discussion, analyst workflow demonstration, and evidence-led review conversations. It should not be positioned as a production fraud platform or autonomous operational decision-maker.
